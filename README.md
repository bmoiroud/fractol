# Fract'ol

## Project Overview

This project is meant to create fractals

<img src="./img/img.png"/>

## Installation

``` bash
$> git clone https://github.com/bmoiroud/fractol.git && cd fractol && make
```

## usage

``` bash
$> ./fractol <fractal_number>
```

| Key         | Action         |
|:-----------:|:---------------|
| H           | help           |
| R           | reset view     |
| Esc         | exit           |
| Space       | modify fractal |
| Arrows      | move around    |
| Mouse Wheel | zoom in/out    |
